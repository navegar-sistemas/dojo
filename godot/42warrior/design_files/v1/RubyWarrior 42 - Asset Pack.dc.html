// GODOT · dica de import & integração
Import PNG → desligue Filter e Mipmaps (Texture → Nearest). Escale por inteiros (2x / 3x) pra manter pixels nítidos. Overlay glitch: CanvasLayer com blend Add/Screen, tile 64px. Sprites/tiles 32×32, ícones 16×16, todos múltiplos de 16. O README.md no .zip traz lista de arquivos, paleta e mapa de assets pro registro data-driven.
